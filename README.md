**Responsive Webpage**

## A modern fully responsive website built using HTML and CSS.
## It adapts smoothly across mobile, tablet, and desktop devices using Flexbox, Grid, and Media Queries.

**Features**
- Fully responsive layout
- Mobile-friendly navigation menu (hamburger menu)
- Hero section with modern design
- Responsive cards layout
- Sidebar with quick links
- CSS Grid + Flexbox used
- Clean UI with modern styling
- Smooth media query breakpoints


**Tech Stack**
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
